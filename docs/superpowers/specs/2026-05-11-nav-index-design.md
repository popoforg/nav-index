# 导航主页设计规格

## 概述

一个可自定义的浏览器导航主页，支持分类管理、卡片布局、拖拽排序，以及配置的导入/导出。采用粗野主义视觉风格。

## 设计方向

- **视觉风格**: 粗野主义 Brutalist
  - 大胆撞色：鲜明饱和的色彩对比
  - 原始边框：粗边框、硬转折、几何感
  - 显眼的网格线：结构清晰可见
  - 高对比度文字：清晰锐利
  - 网格背景纹理增强结构感

## 数据模型

```js
// 分类
Category {
  id: string,          // 唯一标识
  name: string,       // 分类名称
  icon: string,       // 图标 (emoji 或 SVG path)
  order: number,      // 排序
  createdAt: number   // 创建时间戳
}

// 卡片
Card {
  id: string,         // 唯一标识
  categoryId: string, // 所属分类
  title: string,      // 标题
  url: string,        // 链接地址
  emoji: string,      // 图标 emoji
  size: '1x1' | '1x2' | '2x1' | '2x2',  // 卡片尺寸
  color: string,      // 颜色标识
  order: number,      // 同分类内排序
  createdAt: number
}

// 完整配置
Config {
  version: string,    // 配置文件版本
  categories: Category[],
  cards: Card[],
  settings: {
    theme: object      // 预留主题设置
  }
}
```

## 功能模块

### 1. 侧边栏

- **Logo**: 固定在顶部
- **分类列表**: 可拖拽排序，支持编辑/删除
- **添加分类按钮**: 在列表底部
- **导入/导出按钮**: 在侧边栏底部
- **分类悬停**: 显示操作图标（编辑/删除）

### 2. 卡片网格

- **网格系统**: CSS Grid，auto-fill 自适应
- **卡片尺寸**:
  - `1x1`: 默认尺寸
  - `1x2`: 竖向占两格高度
  - `2x1`: 横向占两格宽度
  - `2x2`: 占用 2x2 区域
- **拖拽排序**: 同分类内卡片可拖拽排序
- **右键菜单**: 打开/布局/编辑/删除

### 3. 分类管理

- **添加分类**: 输入名称、选择图标
- **编辑分类**: 修改名称、图标
- **删除分类**: 确认后删除（同时删除该分类下所有卡片）
- **拖拽排序**: 侧边栏内拖拽调整分类顺序

### 4. 导入/导出

- **导出**: 生成包含所有分类和卡片的 JSON 文件，触发下载
- **导入**: 选择 JSON 文件，验证格式，合并或覆盖现有数据
- **文件格式**: `.json` 扩展名

### 5. 搜索栏

- 键盘回车跳转百度搜索

## 交互细节

### 卡片操作
- **左键点击**: 打开链接（新标签页）
- **右键菜单**: 显示操作选项
- **拖拽**: 同一分类内拖拽排序

### 分类操作
- **单击选中**: 切换显示该分类的卡片
- **悬停**: 显示编辑/删除图标
- **拖拽**: 侧边栏内拖拽排序

### 编辑弹窗
- 模态弹窗，居中显示
- 点击遮罩层或取消按钮关闭
- 保存前验证必填项

## 视觉规格

### 色彩系统
```css
--bg-primary: #0a0a0f;         /* 主背景深色 */
--bg-secondary: #14141a;        /* 卡片/侧边栏背景 */
--border-color: #ffffff;        /* 粗野边框 */
--text-primary: #ffffff;
--text-secondary: rgba(255,255,255,0.6);
--accent-red: #ff3b3b;
--accent-yellow: #ffcc00;
--accent-blue: #007aff;
--accent-green: #34c759;
--accent-orange: #ff9500;
--accent-purple: #af52de;
--accent-pink: #ff2d55;
```

### 字体
- 标题: "Outfit", sans-serif (Google Fonts)
- 正文: "Noto Sans SC", sans-serif (Google Fonts)

### 卡片样式
- 背景: 深色纯色 + 左侧彩色边框
- 边框: 3px solid white（粗野风格）
- 圆角: 4px（轻微圆角）
- 悬停: 背景变亮 + 轻微上移

### 动画
- 悬停过渡: 150ms ease-out
- 模态弹出: scale 0.95 → 1, 200ms ease-out
- 卡片拖拽: 跟随鼠标，阴影加深

## 技术实现

- **文件**: 单个 `index.html`，内联 CSS 和 JavaScript
- **存储**: localStorage 持久化配置
- **拖拽**: 原生 HTML5 Drag and Drop API
- **无构建步骤**: 直接浏览器打开即可运行
